How to get your data

Best path for normal Claude Code users is ccusage. Organization admins can also use Anthropic's Admin Usage and Cost APIs.

Claude Code local export
npx ccusage@latest claude daily --json > daily.json
npx ccusage@latest claude session --json > sessions.json
npx ccusage@latest claude monthly --json > monthly.json
Unified agent export
npx ccusage@latest daily --json > usage-daily.json
npx ccusage@latest session --json > usage-sessions.json
Anthropic Admin API

Run the API request locally with an Admin API key, save the JSON, then import the file here. This page does not ask for API keys.
